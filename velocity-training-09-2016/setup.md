## Training Repo

1. Launch Bash Shell (or SSH into the Linux VM)

1. Clone the Training Repo

    ```
    $ mkdir -p ~/workspace
    $ cd ~/workspace
    $ git clone https://github.com/mesosphere/training
    $ cd training
    ```

    For the rest of the training, the training directory will be referred to as `$DCOS_HOME`.
